# Version History

This roadmap summarizes the intended development path for the low-carbon building optimisation project.

| Version | Name | Purpose |
| --- | --- | --- |
| V1 | Base Case Model | Build the baseline decision-support pipeline comparing grid electricity, renewable electricity purchase, rooftop solar PV, and hybrid strategies. |
| V2 | Physical Realism | Add more realistic electricity and solar behaviour, including hourly load, hourly PV generation, grid import/export, and annual target compliance after PV degradation. |
| V3 | PV + ESS Model | Extend the model to include battery energy storage, dispatch logic, self-consumption improvement, and storage economics. |
| V4 | Optimisation Model | Move from predefined strategy comparison to optimisation across PV size, battery size, renewable purchase volume, cost, emissions, and feasibility constraints. |
| V5 | Decision Support Model | Add policy scenarios, sensitivity analysis, stakeholder-specific outputs, and clearer strategic recommendation logic. |
| V6 | Portfolio-Ready Final Project | Package the project into a polished, validated, documented final version suitable for GitHub, portfolio review, and interview discussion. |
